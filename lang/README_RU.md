# Команды & Разрешения

|    COMMAND    |         DESCRIPTION         |  PERMISSION  |
|:-------------:|:---------------------------:|:------------:|
|    `/fap`     |  Главная команда (Фапать).  |  `fap.use`   |
| `/fap reload` | Перезагрузить конфигурацию. | `fap.reload` |
|    `/pip`     |       Размер шланга.        |  `fap.pip`   |
|               |    Иммунитет к задержке.    | `fap.bypass` |
